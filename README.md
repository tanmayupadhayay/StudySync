# 📚 StudySync

StudySync is a modern student productivity dashboard built using **HTML, CSS, and JavaScript**. It helps students organize their study schedule with an easy-to-use interface.

## ✨ Features

- 📊 Dashboard with statistics
- ✅ Task Manager
- 📝 Assignment Tracker
- 📚 Subject Manager
- ⏱️ Pomodoro Timer
- 🌙 Dark Mode
- 💾 Local Storage (data is saved automatically)

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Local Storage API

## 🚀 How to Run

1. Clone the repository
2. Open `index.html` in your browser
